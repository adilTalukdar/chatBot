# ChatBot (Spring Boot)

This is a simple chatbot backend built using Spring Boot.

## 🔧 Tech Stack

* Java
* Spring Boot
* Maven

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/adilTalukdar/chatBot.git
```

2. Go to project folder:

```
cd chatBot
```

3. Build the project:

```
mvn clean package
```

4. Run the application:

```
java -jar target/chatBot-0.0.1-SNAPSHOT.jar
```

## 🌐 Deployed URL

```
https://chatbot-cnm4.onrender.com/webhook
```

## 📦 Project Info

* This is a backend-only project
* No frontend or database is used
* API can be tested using Postman
